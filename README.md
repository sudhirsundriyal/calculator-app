# Java Calculator App with CI/CD (Jenkins + AWS)

## 📌 Project Overview

This project demonstrates a simple Java-based calculator application integrated with a CI (Continuous Integration) pipeline using Jenkins on AWS.

## 🚀 Tech Stack

* Java
* Jenkins
* GitHub
* AWS EC2

## ⚙️ Features

* Basic calculator operations (Add, Subtract, Multiply, Divide)
* Source code managed using GitHub
* Automated build process using Jenkins
* CI pipeline triggered on code changes

## 🛠️ CI/CD Workflow

1. Developer pushes code to GitHub repository
2. Jenkins detects changes (via webhook or polling)
3. Jenkins triggers build job
4. Application is compiled and build is verified
5. Build status is monitored through Jenkins dashboard

## 🖥️ Jenkins Setup

* Jenkins installed on AWS EC2 instance
* Configured Freestyle Job for build automation
* Integrated GitHub repository with Jenkins
* Enabled automatic build triggers
